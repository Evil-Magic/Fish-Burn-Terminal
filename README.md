## ✨ 功能速览
- 🔐 **AES 加密**：可选开关，一键生成加密阅后链接  
- ⏳ **多单位焚毁**：支持秒 / 分 / 时 / 天 / 次 / 人数控制  
- 🔄 **自动焚毁**：倒计时结束后自动清空并淡出动画  
- 🌒 **黑金终端风格**：全黑背景 + 金色发光 UI  
- 📦 **纯前端运行**：无需服务器，支持 GitHub Pages  
- 💾 **缓存销毁**：localStorage 自动清理
